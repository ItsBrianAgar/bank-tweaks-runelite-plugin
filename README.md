# Bank Tweaks

A lightweight RuneLite plugin that helps you customise your bank UI

## Features

### Worn Items Button Position
Move the "Worn Items" (equipment) button in the bank interface to the left or right side of the bank window.

### Custom Bank Tab Names
Personalize your bank tabs with custom names and colors:
- Rename bank tabs with your own labels
- Choose individual colors for each tab name
- Toggle custom names on/off without losing your settings
- This plugin is complemented by the [Mouse Tooltips Plugin](https://github.com/runelite/runelite/wiki/Mouse-Tooltips)

## Configuration

Access the plugin settings through RuneLite's configuration panel:

| Setting | Description | Default |
|---------|-------------|---------|
| Worn Items Button Position | Position of the equipment button (Left/Right) | Right |
| Enable Custom Tab Names | Toggle custom tab names on/off | Off |
| Tab 1-9 Color | Color picker for each tab's name | Orange (#ff9040) |
| Tab 1-9 Name | Custom text for each tab (leave empty to use default) | Empty |
